DAY-3

On 27 March, day-3 of Digital FABLAB, we focused on practical applications of HTML tags we learned previously. We had hands-on practice with various tags, with special emphasis on the img tag. We learned how to:

  • Insert images using the img tag with src, alt, width, and height attributes
  • Create responsive images
  • Position images within the page
  • Create image galleries
  • Use figure and figcaption tags

We also practiced creating forms with input fields, select dropdowns, and buttons. The practical session helped reinforce our understanding of HTML structure and elements.

DAY-4

On 28 March, day-4 of Digital FABLAB, we were introduced to the basics of CSS (Cascading Style Sheets). Our mentor explained the three ways to include CSS in HTML:

  1. Inline CSS - using the style attribute
  2. Internal CSS - using the style tag in the head section
  3. External CSS - linking an external .css file

We learned about CSS selectors, properties, and values. The basic CSS properties covered included:

  • color - for text color
  • background-color - for element backgrounds
  • font-family - for text fonts
  • font-size - for text size
  • margin - for space outside elements
  • padding - for space inside elements
  • border - for element borders

We created our first styled webpage applying these concepts.

DAY-5

On 29 March, day-5 of Digital FABLAB, we continued our CSS learning and were introduced to JavaScript. In the first half, we explored advanced CSS concepts:

  • CSS Box Model (margin, border, padding, content)
  • Flexbox layout
  • CSS Grid basics
  • Responsive design with media queries
  • CSS transitions and animations

In the second half, we had our first JavaScript tutorial. We learned:

  1. How to include JavaScript in HTML (inline, internal with script tags, external .js files)
  2. Basic JavaScript syntax
  3. Variables and data types
  4. Operators
  5. Conditional statements (if-else)
  6. Simple DOM manipulation to change HTML content

We created a simple interactive webpage that responded to user clicks, which helped us understand how HTML, CSS, and JavaScript work together in web development.
